# React Test 

Existem diversas bibliotecas de testes automatizados para serem utilizadas em aplicações React.

O React Test foi focado na RTL (React Testing Library). É mais simples de usar!!

As vantagens da sua utilização são:

Tem muito menos caveats, situações que podem causar pequenos problemas e dores de cabeça inesperadas;

Reforça o bom uso das melhores práticas de testes ao incentivar e facilitar o teste de comportamentos, e o não de implementação;

Permitir a refatoração da sua arquitetura de componentes;

Esse projeto exigiu conhecimento de Mocks, Inputs. Mocar funções em Jest, modulos e funções.
Fizemos também testes de API  e React Router.
